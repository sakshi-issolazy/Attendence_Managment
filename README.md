# Attendance Manager
Attendance Manager application for Android developed using Java, SQLite and Shared Preferences.

## Author
- Nikhil Kumar

## Features
- Add Student
- View Student
- Add Faculty
- View Faculty
- Manage Attendance

## Usage
- Admin Username : `admin`
- Admin Password : `pass`
